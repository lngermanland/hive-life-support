# Поддержка HiveLife

⚠️ **Важно:** Это экспериментальный проект, созданный для обучения. Пожалуйста, не ожидайте от него слишком многого — здесь могут быть баги, а некоторые функции могут работать не идеально. Но я старался! 😊

## 📱 О приложении
HiveLife — это удобный помощник для поиска ресторанов, кафе, магазинов, парковок и достопримечательностей рядом с вами. Приложение использует данные Apple Maps и полностью бесплатно.

## ❓ Часто задаваемые вопросы

### Как работает поиск мест?
Приложение показывает места в выбранном радиусе от вашего текущего местоположения. Вы можете фильтровать результаты по категориям.

### Почему я не вижу некоторые места?
Apple Maps может отображать не все места в некоторых районах. Попробуйте увеличить радиус поиска или выбрать другую категорию.

### Как добавить место в избранное?
Нажмите на сердечко рядом с названием места. Все избранные места сохраняются в профиле.

### Как построить маршрут?
Нажмите на любое место, затем выберите "Построить маршрут". Приложение откроет маршрут в Apple Картах.

### Приложение платное?
Нет, HiveLife полностью бесплатное и не требует регистрации.

### Какие данные собирает приложение?
Приложение запрашивает только ваше текущее местоположение для поиска мест рядом с вами. Эти данные не передаются третьим лицам.
